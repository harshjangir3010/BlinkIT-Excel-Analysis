# Blinkit Grocery Sales Analysis — Excel

## 📊 Project Overview
Performed end-to-end data analysis on Blinkit 
grocery dataset using Microsoft Excel.

## 📁 Dataset
- Source: Kaggle
- Records: 8523 rows
- Columns: 12

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers

## 🔍 Key Insights
- Fruits & Vegetables top selling category
- Medium outlets generate highest sales
- Tier 3 cities most profitable
- Customers prefer Low Fat products
- 2018 was peak year for outlet expansion
